geom2d

A simple 2D geometry package that includes point, line, arc, ellipse, cubic bezier, and miscellaneous implementations of computational geometry solutions.

All objects (points, lines, etc) are implemented as immutable tuples.

Apologies/caveats/etc.

Testing is artisanal at best.

Any math errors are my fault alone and, while the library is pretty fast, there may be obvious optimizations that I’ve overlooked or ignored for the sake of clarity (to me).
